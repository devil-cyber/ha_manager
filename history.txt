Your question is a bit broad. Here are some common commands to check Ubuntu system information:

Check Ubuntu Version
lsb_release -a

or

cat /etc/os-release

or

hostnamectl
Check Kernel Version
uname -r
Check System Architecture
uname -m
Check CPU Information
lscpu
Check Memory (RAM)
free -h
Check Disk Usage
df -h
Check Running Ubuntu Release
cat /etc/issue

Example output:

Ubuntu 20.04.6 LTS \n \l
